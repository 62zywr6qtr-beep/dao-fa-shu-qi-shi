# dao-fa-shu-qi-shi
# 道法术器势 · 元框架

> 万物皆可套用的系统分析框架。任何系统，从任意维度切入，都可以用这五层来拆解。

## 五层结构

| 层级 | 核心问题 | 回答模板 |
|---|---|---|
| **道** | 这东西的本质是什么？ | "X的本质是Y" |
| **法** | 什么规则在约束它？ | "X遵循Y规则" |
| **术** | 具体怎么做？ | "做X需要Y步骤" |
| **器** | 用什么工具实现？ | "X用Y工具" |
| **势** | 现在处于什么趋势/时机？ | "X正在Y趋势中" |

## 核心原则

- **纵向一致**：上一层决定下一层，不能矛盾
- **横向关联**：不同系统的同一层可能有共享规则
- **跨层影响**：势的变化可以影响道，反向亦然

## 快速锚定

| 层级 | 锚定问题 |
|---|---|
| 道 | 如果这个系统只能保留一个规律，是什么？ |
| 法 | 什么规则让这个系统不会崩溃？ |
| 术 | 要做好这件事，有哪几步？ |
| 器 | 没有工具的情况下，这事能做吗？ |
| 势 | 现在做这件事，时机对吗？ |

## 使用方法

1. **先定道** — 找到核心不变的本质
2. **再立法** — 找出约束规则
3. **再出术** — 拆解具体步骤
4. **再选器** — 选择实现工具
5. **再观势** — 判断当前时机
